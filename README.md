# Devbook em GO

Site de publicação, projeto de base para estudos do go

## Clone do Projeto

```
$ git clone git@github.com:agnaldopidev/devbook.git
$ cd devbook
```

## Docker

```
docker compose up -d

```

## Rodar os scripts sql na pasta api/sql

## Execução do Projeto

```
$ cd api
$ go build
$ ./api
$ cd wepapp
$ go build
$ ./wepapp
```

## Acesso

```
http://localhost:3000

```
